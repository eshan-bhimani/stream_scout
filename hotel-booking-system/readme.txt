Technologies (course + 3rd party)
---------------------------------
- Java 17, Spring Boot 3.2 (spring-boot-starter-web, thymeleaf, jdbc, security)
- Maven
- MySQL 8 (JDBC driver: com.mysql.cj.jdbc.Driver)
- Thymeleaf + thymeleaf-extras-springsecurity6
- Server-side data access: Spring JdbcTemplate with PreparedStatement (? placeholders) in repository classes

No extra third-party libraries beyond Spring Boot / MySQL driver (no permission email required).
